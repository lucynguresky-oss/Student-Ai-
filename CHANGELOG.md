# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2024-XX-XX

### Added
- Monorepo scaffold (Turborepo + pnpm workspaces)
- NestJS API with Fastify adapter, health endpoints, OpenAPI docs
- Prisma schema (users, profiles, roles, posts, learning, library, papers, AI, payments)
- Next.js web app with Learnix design system
- Next.js admin dashboard with sidebar navigation
- Expo mobile app scaffold with bottom tabs
- Docker Compose (PostgreSQL, Redis, Meilisearch, Mailhog)
- Shared packages (types, SDK, analytics, feature-flags, RAG)
- Database seed script (5 subjects, 20 topics, 5 users, 20 lessons, 8 badges)
- ADRs 001-010
